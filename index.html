Human Gesture Recognition with MS Kinect

Human body gesture recognition with Kinect using SSS features

View on GitHub Download .zip Download .tar.gz

Implemented the gesture recognition with Kinect using SSS features algorithm explained in the paper: http://cs.adelaide.edu.au/~qsheng/papers/p23-zhao.pdf

A sample video showing result of the trained gesture recognition can be found at: https://www.dropbox.com/s/jf4zfwk3o26geyv/handsup_duck_wave_3each_100lambda_5nclust.mp4?dl=0 Sample Video
